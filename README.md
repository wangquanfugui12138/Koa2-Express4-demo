# Koa2 & Express4 Demo

## 使用方法

```bash

git clone git@github.com:wangquanfugui12138/Koa2-Express4-demo.git

```

#### 本地调试

```bash

yarn || npm i

yarn start || npm start --启动Client 

yarn express || yarn koa --选择启动koa或express的Server

localhost:3000 

```

#### 远程调试

```bash

yarn || npm i

yarn build || npm build --打包

yarn express || yarn koa --选择启动koa或express的Server

localhost:3001

```
